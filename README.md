# Custom-dns-server
A simple DNS server with caching and forwarding

Project Description:
This project implements a simple DNS server using Python.
It resolves domain names using:

Local DNS table
Caching mechanism (TTL)
Forwarding to external DNS (Google DNS)

Features:
Local domain resolution
DNS caching with TTL
Forwarding to external DNS server
UDP-based communication

Technologies Used:
Python
Socket Programming
